# Ecological Footprint
Ecological footprint (abbreviated EF) :
=>scoring system :
- score points are just a scaled number based on assigned weights to every response that increases when the the ecological footprint is worse (similar to the quiz in "https://www.environnement.gouv.qc.ca/jeunesse/jeux/questionnaires/empreinte/questionnaire.htm"). 

=> score range between ≈ [58~631] (the less the better EF)

=> Global Hectares (gha) and number of Earths are calculated based on the average EF of Tunisia (1.9gha as of 2022, data taken from https://data.footprintnetwork.org) and adjusted by a scaling factor : Scaling factor = Known national EF​  / App avg score, every response is weighed as a number between [0-1] based on strengh of impact on ecological footprint, and the result is aggregated to get the final score.
=>Because we're enforcing that the Average user in our app ≈ Average Tunisian in reality we stay statistically aligned with reality, even if it's simplified.
=> used in this work :
- data from : https://www.footprintnetwork.org/
